## 技术栈

vue2 + vuex + vue-router + vue-cli + axios + element-ui + echarts

## 依赖项版本

"axios": "^0.18.0"  
"echarts": "^4.1.0"  
"element-ui": "^2.5.4"  
"vue": "^2.6.6"  
"vue-router": "^3.0.1"  
"vuex": "^3.1.0"

# 构建流程

``` bash
# 安装依赖项（初次运行）
npm install

# 调试运行
npm run dev

# 项目发布
npm run build

```
